CSS Reference
CSS Selectors
Selector Name Example Description
Element body selects <body> All specific HTML elements.
ID #hello selects <a id="hello> Elements with the specified ID, these value's are all unique.
Class .hello selects <a class="hello"> Elements with the specified class, these can repeat.
Attribute img[src] selects <img src="helloimage.png"> but not <img> Elements with the specified attribute.
Pseudo-class a:hover selects <a> but only when the cursor is hovering over the link. Specified element but only in a certain state. ie. a cursor hovers over a link.
